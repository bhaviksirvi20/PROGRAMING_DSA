Programming Concepts Q&A

What is programming?

A set of Instructions for computer to perform specific tasks and solve problems.

What is a translator in programming?

Translator ek software hota hai jo high-level code ko machine code mein convert karta hai.

Types of translators in C and C++

Compiler ek software hota hai jo pura source code ek sath machine code mein convert karta hai.

Interpreter ek software hota hai jo line-by-line code execute karta hai, lekin C/C++ mein use nahi hota.

Difference between Compiler and Interpreter

Compiler jo pura source code convert karta hai, Interpreter line-by-line code execute karta hai.
